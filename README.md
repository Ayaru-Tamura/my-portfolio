# My Portfolio

Astro を使用して制作しているポートフォリオサイトです。

## URL
[公開サイトはこちら](https://my-portfolio-five-lac-85.vercel.app/)

## 概要
このサイトでは、自己紹介、スキル、制作物、連絡先を掲載しています。  
受付事務・音響・照明の現場経験と、ITサポート・検証業務の経験をもとに、  
業務改善につながる視点を意識して制作しています。

## 使用技術

- HTML / CSS
- JavaScript
- Vercel
- GitHub
